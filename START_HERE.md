## ⚡ TROUBLESHOOTING

| Problem | Fix |
|---------|-----|
| Map won't load | Refresh browser + check internet |
| Port in use | Kill existing process or use different port |
| Backend 404 | Make sure backend is running on 8000 |
| No Mapbox token | Get free token at mapbox.com |
| Scenario won't draw | Click Cancel first, then try again |
| Compare crashes | Reload page |
